# `@meatwallace/commitlint-config`

personal [commitlint](https://commitlint.js.org) configuration

## Usage

**Install**:

```sh
$ yarn add --dev @meatwallace/commitlint-config
```

**Extend your `commitlint` configuration file**:

```js
// commitlint.config.js
module.exports = {
  extends: ['@meatwallace'],
  rules: {
    // overrides
  },
};
```
