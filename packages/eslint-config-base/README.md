# `@meatwallace/eslint-config-base`

personal [ESLint](https://eslint.org) base configuration

## Usage

**Install**:

```sh
yarn add --dev @meatwallace/eslint-config-base
```

**Extend your `eslint` configuration file**:

```js
// eslint.config.js
module.exports = {
  extends: ['@meatwallace/eslint-config-base'],
  rules: {
    // overrides
  },
};
```
